# CodeCritters

Pokemon-style roguelike for the terminal. Every critter is a programming concept.

Built on [Vexel](https://github.com/dkremer/vexel), a terminal game engine using the Kitty graphics protocol.

## Running

```bash
# Install Vexel, then:
vexel run .
# or from the Vexel repo:
zig build run -- /path/to/codecritters-game/
```
